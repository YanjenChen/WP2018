Augmented Reality Tower Defense

This idea is inspired from Paper Cubes, a goolge experiment. The author uses cubes as marker to control stick figures' behavior in AR enviroment. I think Paper Cubes is a great idea, it's fun to use hand-making component to interact with AR, and it have the potential to extend to more application (that's why i want to propose this idea). The author use Unity and Vuforia to make this project, but I think it MAYBE can use web-AR to achieve similar result.

2018.9.26 Update
After discuss with professor, the following are some improvement tip to make this idea more fun and executable:

  • Action of stick figure is FUCKING HARD to implement in web AR, its Unity's strength. Using static object to replace stick figure may be a good idea.
  • Using chessboard or grid in reality can simply programming and game process design. Maybe can turn to design a board game.
  • Add gesture or facial detect ability. Using super skill in game after detected specific gesture.
  • Use marker to detect main plane.

Tech, library or framework may help achieve this idea

  • WebXR. Under development google immersive web API.
  • WebAR. A new API under WebXR.
  • AR.js - Augmented Reality for the Web. Marker based JS AR library (but seems like marker base system can not use code to control object's coordinate...).
  • Create your Own Marker. It's possible to use customized marker in AR.js.
  • A-Frame. An entity component system(ECS) framework for Three.js, can intergrate with AR.js.
